# Spec: Portfolio Expansion

## MODIFIED Requirements

### Requirement: Homepage Hero Structure
The homepage MUST have a primary greeting and name section OUTSIDE the bento grid.

#### Scenario: Hero prominence
The hero section is the first element below the (optional) nav and spans the full width of the container, distinct from the bento grid.

### Requirement: Timeline Visualization
The site MUST include a vertical timeline highlighting work history.

#### Scenario: Heimdall Focus
The timeline explicitly highlights "Heimdall" as the primary effort from 2025 to Present.

### Requirement: Technical Skills Display
The site MUST display a curated list of technical skills using icons.

#### Scenario: Icon Library
The skills section uses DevIcons for React, Tailwind, Docker, etc.

## ADDED Requirements

### Requirement: Project Archive
The site MUST provide a dedicated listing of all projects.

#### Scenario: Projects URL
Navigating to `/projects` shows a list of all items in the `projects` content directory.

### Requirement: Detailed About Page
The site MUST have a recruiter-focused `/about` page.

#### Scenario: Scaled Components
The timeline and skills displays are larger and more detailed on the `/about` page than on the homepage.
