# Spec: Design Polish

## ADDED Requirements

### Requirement: Atmospheric Depth
The application MUST display visual depth using gradients and lighting effects.

#### Scenario: Background Vignette
The page background is not a flat solid color but includes a subtle radial gradient or "spotlight" effect to focus attention.

#### Scenario: Card Surface
Cards have a subtle gradient fill (e.g. top-down lighting) rather than a solid flat color.

### Requirement: Interactive Polish
Interactive elements MUST respond with fluid visual cues.

#### Scenario: Card Hover
Hovering over a Bento card triggers a visual state change (e.g., border brightening or slight lift).

### Requirement: Hero Distinction
The Hero section MUST be visually distinct from other grid items.

#### Scenario: Dynamic Gradient
The Hero card features a more vibrant or complex background gradient than standard content cards.
